General Rules for CSS:

  • All CSS(Cascading Style Sheet) files are saved with a ".css" extension.
  • To link external CSS files, you must use a tag with an href set to the CSS file.
  • CSS files use classes to give properties to. you must use a "." + className
  • Classes within CSS must have curly brackets {} to enclose their properties.
  • Some HTML tags can be given properties. For example, the < form > tag can be referenced with .form {Insert properties here}
  • Others, you can name yourself. Such as our navigation bar, which we reference as .navbar
  • CSS can specify colors, size, location, and actions of given classes
  • Values for assigning locations can be given as a percentage or as pixels.
