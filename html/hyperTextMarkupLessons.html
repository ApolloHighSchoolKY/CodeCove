General Rules for HTML:

  • All HTML (Hyper Text Markup Language) Files are saved with a ".html" extension.
  • Every element within your webpage must be enclosed in tags
  • Some examples of elements include: "html", "body", "ul", "li", "form", "title", "header", "footer"
  • *Almost* every element must also be closed with "<" + "/" + ">" tags
  • These exceptions are known as 'Singleton' tags and include the "link", "br", "col", "command", and "embed" tags
  • In 'Singleton' tags, it is not necessary to include a closing tag ("<" + "/" + ">")
  • HTML is compatible with many different languages, one including CSS, which makes this text blue.
