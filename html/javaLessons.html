Welcome to the Java Lessons Page

General Rules for Java:

  • Java Files are saved with a ".java" extension.
  • All code in Java must be run inside of a class, which must match the name of the file (Such a HelloWorld in HelloWorld.java)
  • Java is a language that is used by more than 3 billion devices. As such, it is an important language to learn for any person thinking of writing a single or multi-platform program.
  • Java has many important aspects to learn of, from classes, to values, to everything in between. That's what this section is about!
