*Prior HTML Lessons Recommended* General Rules for Javascript:

  • All Javascript Files are saved with a ".js" extension.
  • When used in .html files, javascript syntax must be preceded by "javascript:" or enclosed by a "script" tag.
  • Javascript is very similar to Java, and is an official programming languge which allows data manipulation.
  • Functions in javascript must have "{}" following the "function" + name of the function.
  • Javascript functions (including default ones) will highlight this color.
  • Default types of objects (such as function and var) will highlight this color.
